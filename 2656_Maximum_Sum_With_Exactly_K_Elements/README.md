### 2656.K个元素的最大和

给你一个下标从 **0** 开始的整数数组 `nums` 和一个整数 `k` 。你需要执行以下操作 **恰好** `k` 次，最大化你的得分：

1. 从 `nums` 中选择一个元素 `m` 。
2. 将选中的元素 `m` 从数组中删除。
3. 将新元素 `m + 1` 添加到数组中。
4. 你的得分增加 `m` 。

请你返回执行以上操作恰好 `k` 次后的最大得分。

来源：力扣

https://leetcode.cn/problems/maximum-sum-with-exactly-k-elements/



思路：

​		本题其实在求一个以数组初始的最大值为首项，公差为1的`k`项等差数列的和：`k*(maxv+maxv+k-1)/2`。
